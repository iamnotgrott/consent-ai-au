# Consent OS (IQ Case #1)

Web prototype: unified consent dashboard, data-flow map, permission history, and revoke flow with demo data.

## Scripts

```bash
npm install
npm run dev      # local development
npm run build    # production build to dist/
npm run preview  # serve dist/
npm run lint
```
